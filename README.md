# CarPark
用停车场管理系统来练习栈和队列
------------------------------------------------------
停车场是一个可以停放n辆汽车的狭长通道，且只有一个大门可供汽车进出。停车场内如果已经停满，后来的汽车只能在门外的便道上等候，一旦有车开走，排在通道上的第一辆车即可开入；当停车场内的每辆车要离开时，在它之后进入的车辆必须先退出停车场为期让路，待该辆车开出大门，其他车辆再按原次序进入停车场费。为停车场编写按上述要求进行管理的模拟程序。
